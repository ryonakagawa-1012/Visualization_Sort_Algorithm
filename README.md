# Visualization_Sort_Algorithm
様々なソートアルゴリズムをHandy Graphic を用いて可視化する

## 実装したソートアルゴリズム
- バブルソート
- 選択ソート
- 挿入ソート
- マージソート
- ボゴソート

## 実行方法
1. Visualization_Sort_Algorithm内に移動した後、コンソール上で以下のコマンドを実行
    ```
    hgcc main.c func_main.c func_sub.c && ./a.out
    ```
2. 画面上にソートアルゴリズムの選択肢が表示されるので、数字を入力
3. ソートする長さを入力
4. ソートアルゴリズムが実行される
